# 苍蝇小馆

> 土菜馆发现与分享平台

## 项目简介

苍蝇小馆是一款面向广大吃货的小程序，用于发现、记录和分享那些藏在街头巷尾的土菜馆。

## 技术栈

| 模块 | 技术 |
|------|------|
| 小程序前端 | uni-app |
| 管理后台前端 | Vue 2 |
| 后端 | Java 8 / Spring Boot 2.7.x |
| 数据库 | MySQL 5.7 |
| 缓存 | Redis |
| 存储 | 腾讯云 COS |
